# Баг-репорт: корзина не привязана к аккаунту и не синхронизируется между устройствами

## Краткое описание

Корзина хранится локально в браузере и не зависит от авторизации пользователя.

При переключении между аккаунтами в одном браузере корзина не меняется.  
При входе под тем же аккаунтом на другом устройстве корзина не подтягивается.

## Окружение

- Web
- Тестовое окружение
- Desktop / Mobile
- Browser: Chrome / Safari

## Предусловия

- Есть два аккаунта: аккаунт A и аккаунт B.
- Пользователь может авторизоваться и выходить из аккаунта.
- В корзину можно добавлять товары.

## Шаги воспроизведения

1. Авторизоваться под аккаунтом A.
2. Добавить товар в корзину.
3. Выйти из аккаунта.
4. Авторизоваться под аккаунтом B в том же браузере.
5. Проверить корзину.
6. Авторизоваться под аккаунтом A на другом устройстве.
7. Проверить корзину.

## Фактический результат

Корзина не меняется при переключении между аккаунтами в одном браузере.

Корзина не синхронизируется между устройствами.

Корзина хранится локально и не привязана к пользователю.

## Ожидаемый результат

Корзина авторизованного пользователя должна быть привязана к аккаунту.

Корзина должна:

- синхронизироваться между устройствами;
- изменяться при переключении аккаунтов;
- сохраняться между сессиями;
- подтягиваться при повторной авторизации.

## Дополнительные требования к логике

Для неавторизованного пользователя корзина должна работать как гостевая.

При авторизации гостевая корзина и корзина пользователя должны объединяться.

Если в гостевой и пользовательской корзине есть одинаковые товары, количество должно суммироваться.

При logout пользовательская корзина не должна удаляться.

После logout должна отображаться гостевая корзина, если она есть.

Если гостевой корзины нет, должна отображаться пустая корзина.

При повторной авторизации корзина пользователя должна снова подтягиваться.

## Критерии приемки

- Корзина авторизованного пользователя хранится на сервере.
- Корзина неавторизованного пользователя хранится локально.
- При авторизации выполняется объединение локальной и серверной корзины.
- При переключении между аккаунтами корзина обновляется в соответствии с текущим аккаунтом.
- При входе на другом устройстве корзина пользователя подтягивается.
- При logout серверная корзина пользователя сохраняется.
- После logout отображается локальная гостевая корзина или пустое состояние.
- При повторной авторизации серверная корзина пользователя восстанавливается.

## Влияние

Пользователь может потерять ожидаемый состав корзины при смене устройства или аккаунта.

При использовании одного браузера разными аккаунтами корзина может отображать товары не того пользователя.

Это может привести к ошибкам при оформлении заказа, путанице между аккаунтами и снижению доверия к работе личного кабинета.
