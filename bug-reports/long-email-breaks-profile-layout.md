# Баг-репорт: длинный email не помещается в контейнере и ломает отображение профиля

## Краткое описание

При отображении длинного email в профиле текст не помещается в контейнер и выходит за границы блока.

Строка не переносится, из-за чего нарушается верстка страницы.

## Окружение

- Web
- Тестовое окружение
- Desktop / Mobile
- Browser: Chrome / Safari

## Предусловия

- Есть пользователь с длинным email.
- Пользователь авторизован.

## Шаги воспроизведения

1. Авторизоваться под пользователем с длинным email.
2. Перейти в профиль.
3. Проверить отображение email.

## Фактический результат

Длинный email не переносится.

Текст выходит за границы контейнера.

Верстка страницы нарушается.

## Ожидаемый результат

Email корректно отображается в пределах контейнера.

Длинный текст переносится или обрезается по правилам дизайна.

Верстка страницы не ломается.

## Влияние

Пользователь с длинным email видит некорректное отображение данных в профиле.

Проблема ухудшает внешний вид страницы и может затрагивать адаптивную верстку на небольших экранах.
