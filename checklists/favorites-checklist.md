# Чек-лист тестирования раздела “Избранное”

## Цель

Проверить, что пользователь может открывать раздел “Избранное”, добавлять и удалять товары, видеть актуальный список избранных товаров, а данные сохраняются после перезагрузки страницы.

## Область проверки

- открытие раздела “Избранное”;
- доступность раздела для авторизованного пользователя;
- добавление товара в избранное;
- удаление товара из избранного;
- отображение списка товаров;
- счетчик избранного;
- сохранение данных после перезагрузки;
- работа с товарами “Нет в наличии”;
- переход в карточку товара;
- добавление товара из избранного в корзину.

---

## 1. Страница “Избранное”

- [ ] Переход по иконке “Избранное” открывает раздел “Избранное”.
- [ ] Раздел “Избранное” открывается корректно.
- [ ] При открытии раздела нет ошибок загрузки.
- [ ] При открытии раздела нет ошибок `404`.
- [ ] При открытии раздела нет ошибок `500`.
- [ ] При открытии раздела нет ошибок CORS.
- [ ] Раздел корректно отображается после обновления страницы.
- [ ] Заголовок соответствует выбранному разделу.
- [ ] Все ресурсы раздела загружаются корректно.
- [ ] При загрузке раздела нет визуальных сдвигов контента.
- [ ] Раздел недоступен без авторизации.
- [ ] При отсутствии товаров отображается пустое состояние.

---

## 2. Добавление товара в избранное

- [ ] Товар можно добавить в избранное.
- [ ] Добавленный товар отображается в списке избранного.
- [ ] Добавленный товар не исчезает после перезагрузки страницы.
- [ ] Товар со статусом “Нет в наличии” можно добавить в избранное.

---

## 3. Удаление товара из избранного

- [ ] Товар удаляется из избранного по кнопке “Удалить”.
- [ ] Удаление происходит без перезагрузки страницы.
- [ ] После удаления счетчик на иконке в хедере обновляется.
- [ ] После удаления товар отсутствует в списке избранного.
- [ ] После перезагрузки страницы удаленный товар не возвращается в избранное.
- [ ] Товар со статусом “Нет в наличии” можно удалить из избранного.

---

## 4. Список товаров в избранном

- [ ] Товары в списке избранного отображаются корректно.
- [ ] Информация о товаре не обрезается.
- [ ] Информация о товаре отображается корректно.
- [ ] Данные в избранном соответствуют данным товара.
- [ ] Один и тот же товар не дублируется в списке избранного.
- [ ] Счетчик на иконке в хедере отображает фактическое количество товаров в избранном.
- [ ] Список товаров сохраняется после перезагрузки страницы.
- [ ] Клик по товару открывает страницу товара.
- [ ] При добавлении товара в корзину отображается уведомление.
- [ ] Уведомление о добавлении в корзину отображается у выбранного товара.
- [ ] После добавления в корзину товар остается в избранном.
- [ ] Для товара со статусом “Нет в наличии” в избранном отображается соответствующий статус.
- [ ] Товар со статусом “Нет в наличии” нельзя добавить в корзину.
- [ ] Сортировка товаров в избранном работает корректно.
