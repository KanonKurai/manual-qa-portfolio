# Чек-лист тестирования корзины

## Цель

Проверить, что пользователь может открыть корзину, добавлять и удалять товары, изменять количество, видеть корректные счетчики, суммы и состояние товаров.

## Область проверки

- открытие раздела “Корзина”;
- доступность корзины без авторизации;
- добавление товара в корзину;
- изменение количества товара;
- удаление одного товара;
- очистка корзины;
- отображение списка товаров;
- счетчики товаров;
- товары “Нет в наличии”;
- расчеты и итоговая сумма;
- переход к оформлению заказа.

---

## 1. Страница “Корзина”

- [ ] Переход по иконке “Корзина” открывает раздел “Корзина”.
- [ ] Раздел “Корзина” открывается корректно.
- [ ] При открытии раздела нет ошибок загрузки.
- [ ] При открытии раздела нет ошибок `404`.
- [ ] При открытии раздела нет ошибок `500`.
- [ ] При открытии раздела нет ошибок CORS.
- [ ] Раздел корректно отображается после обновления страницы.
- [ ] Заголовок соответствует выбранному разделу.
- [ ] Все ресурсы раздела загружаются корректно.
- [ ] При загрузке раздела нет визуальных сдвигов контента.
- [ ] Раздел доступен без авторизации.
- [ ] При отсутствии товаров отображается пустое состояние.
- [ ] Кнопка “Перейти в каталог” отображается.
- [ ] Кнопка “Перейти в каталог” ведет в каталог.
- [ ] Раздел “Вам может понравиться” отображается.

---

## 2. Добавление товара в корзину

- [ ] После добавления товара отображается блок управления количеством.
- [ ] В блоке управления количеством отображаются кнопки “+” и “-”.
- [ ] Кнопка “+” увеличивает количество товара на 1.
- [ ] Кнопка “-” уменьшает количество товара на 1.
- [ ] При количестве товара `1` кнопка “-” удаляет товар из корзины.
- [ ] После удаления при количестве `1` блок управления количеством заменяется на кнопку “В корзину”.
- [ ] Счетчик на товаре обновляется.
- [ ] Добавление товара происходит без перезагрузки страницы.
- [ ] Счетчик на иконке корзины в хедере обновляется.
- [ ] Добавленный товар не исчезает после перезагрузки страницы.

---

## 3. Удаление товара из корзины

- [ ] На товаре отображается иконка удаления.
- [ ] По иконке удаления удаляется выбранный товар.
- [ ] Кнопка “Очистить” отображается при наличии товаров в корзине.
- [ ] По клику на кнопку “Очистить” удаляются все товары из корзины.
- [ ] Счетчик на иконке корзины в хедере отображает фактическое количество товаров.
- [ ] Общий счетчик товаров отображает фактическое количество товаров.
- [ ] Удаленный товар не возвращается после перезагрузки страницы.

---

## 4. Список товаров в корзине

- [ ] Товары в корзине отображаются корректно.
- [ ] Информация о товаре не обрезается.
- [ ] Информация о товаре отображается корректно.
- [ ] Данные в корзине соответствуют данным товара.
- [ ] У товара со скидкой отображаются старая цена и актуальная цена со скидкой.
- [ ] Один и тот же товар не дублируется в списке корзины.
- [ ] Счетчик на иконке корзины в хедере отображает фактическое количество товаров.
- [ ] Счетчик на товаре отображает фактическое количество товара.
- [ ] Общий счетчик товаров отображает фактическое количество товаров.
- [ ] Счетчики синхронизированы между собой: товар, общий счетчик, хедер.
- [ ] Список товаров сохраняется после перезагрузки страницы.
- [ ] Клик по товару открывает страницу товара.
- [ ] Для товара со статусом “Нет в наличии” в корзине отображается соответствующий статус.
- [ ] Товар со статусом “Нет в наличии” недоступен для заказа.
- [ ] Для товара со статусом “Нет в наличии” не отображается управление количеством.
- [ ] Сортировка товаров в корзине работает корректно.

---

## 5. Итоги и расчеты

- [ ] Сумма по позиции отображается корректно.
- [ ] Общая сумма отображается корректно.
- [ ] Если у товара есть скидка, она учитывается в итоговой сумме корзины.
- [ ] Итоговая сумма к оплате рассчитывается корректно.
- [ ] При изменении количества товара сумма пересчитывается.
- [ ] После перезагрузки страницы сумма остается корректной.
- [ ] Счетчик товаров в штуках отображает фактическое количество.
- [ ] Кнопка “Перейти к оформлению” кликабельна.
- [ ] Кнопка “Перейти к оформлению” ведет на авторизацию или страницу оформления заказа.
